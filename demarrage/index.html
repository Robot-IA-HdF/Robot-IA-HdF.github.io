Comment débuter votre challenge ?

Équipe

Une équipe doit être mixte (ENSAM/IMT NE) et compter entre 4 et 6 élèves. Vous devez :

  • choisir un <NOM_EQUIPE>
  • désigner un responsable du matériel qui vérifie que tout est présent conformément à la liste suivante et sera charge de restituer tous les éléments à la fin du challenge :
    • Pibot + raspberry pi + lidar 2d
    • PC portable
    • NED2 + kit vision
    • PC fixe
    • switch ethernet + câbles
    • multiprise
  • récupérer auprès des coachs la configuration réseau et les identifiants de vos robots et PCs
  • changer vos mots de passe sur la pi3, le PC portable et PC fixe et s'en souvenir

Github

Travail à faire :

  1. Chaque membre de l'équipe doit avoir un compte sur github
  2. Créer un dépôt git privé (sélectionner la licence libre MIT) portant le nom de l'équipe
  3. Inviter l'ensemble des membres de l'équipe dans ce dépôt
  4. Ajouter un README dans le dépôt avec a minima les informations suivantes: le nom du projet/équipe, les noms des membres, ...
  5. Donner l'URL de votre dépôt git aux organisateurs et les inviter

Sujet du Challenge

Lisez attentivement la section Challenge qui décrit le sujet du challenge. A vous de définir vos priorités et les fonctionnalités innovantes que vous pourriez imaginer.
