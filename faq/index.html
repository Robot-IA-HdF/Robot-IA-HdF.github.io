Frequently Asked Questions

Les ordinateurs portables:

  • Comment je me logge sur les ordinateurs portables ?
    Vous pouvez vous logger sur le compte: ros avec le mot de passe ros.
  • J'ai besoin d'installer un logiciel ou un outil, comment je fais ?
    Vous êtes administrateurs sur les machines. Vous pouvez donc utiliser le gestionnaire de paquets: apt pour des paquets Ubuntu, ROS2 ... ou pip pour des librairies python.
